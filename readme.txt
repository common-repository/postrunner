Postrunner is a guest posting system connecting authors with site owners.  This plugin facilitates the process of hooking a WordPress site into Postrunner to receive guest posts.  Learn more at [http://postrunner.com](http://postrunner.com)

== Changelog ==

= 3.0.8 =
* Fixed xmlrpc file access denial fromsecurity protocals on some hosting providers. 

= 3.0.7 =
* Added "Daily Traffic" count, which is enabled by default.  To disable, look for "Share Traffic Data" near the bottom of the "General" section of "Settings".
* Added ability to blacklist particular authors, to prevent them from sending posts to you again.

= 3.0.6 =
* Fixed a bug that caused post numbers to be inaccurate on the actual PostRunner website.
